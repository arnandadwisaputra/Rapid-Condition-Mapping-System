# Rapid-Condition-Mapping-System

#Membuat sistem yang membantu tim asesmen cepat mencatat kondisi wilayah bencana secara terstruktur.

Mengklasifikasikan kondisi wilayah berdasarkan aturan logika yang jelas.

Menyediakan ringkasan kondisi lapangan untuk pengambilan keputusan manusia.

Menerapkan konsep array, percabangan, perulangan, dan fungsi dalam konteks nyata.
